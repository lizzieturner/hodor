# HODOR

## OBJECTIVES   
An optional project to learn about:   
   * HTTP headers   
   * HTML scraping   
   * solving CAPTCHA with OCR   
   * TOR   
   * proxy lists

## REQUIREMENTS   

   * all exercise must be kept in separate directories   
   * you can use the language of your choice
